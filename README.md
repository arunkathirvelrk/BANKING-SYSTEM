# BANKING-SYSTEM
# 💳 Secure Mobile Banking & Instant Transaction Monitoring System

## 📌 Project Description

This project is a **full-stack banking application** that simulates a real-world mobile banking system. It allows users to securely manage their accounts, perform money transfers, and receive instant notifications.

The system is designed using a **microservices architecture**, where each service handles a specific responsibility such as authentication, user management, accounts, transactions, and notifications.

A key feature of this project is **real-time transaction monitoring with basic fraud detection**. Suspicious transactions (like high-value transfers or rapid activity) are automatically flagged, and users are notified immediately.
